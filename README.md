## Installation

```
$ git clone https://github.com/dreadrocksean/VanillaJSDemo.git
$ git checkout feature/ES6
$ npm i
$ gulp
```

For pure ES5 Javascript and CSS with no bells and whistles omit
```
$ git checkout feature/ES6
```
Or if you have already executed it, do
```
$ git checkout master
```
